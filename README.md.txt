# Car Catalogue Project

A static website built with HTML, CSS, JavaScript, and images.

## Structure
- `HTML_files/` → All HTML pages
- `Css_files/` → Stylesheets
- `JavaScript_files/` → JS scripts
- `Image_files/` → Images

## How to run
Just open any HTML file from `HTML_files/` (e.g., `index.html`) in your browser.
